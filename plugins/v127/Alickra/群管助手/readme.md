# 群管助手 — WAuxiliary 群管插件

微信群管理插件，适用于 WAuxiliary 框架。

## 功能

### 社区版（免费）
- 入群欢迎词（支持 `{昵称}` 变量）
- 违禁词过滤（自动提醒）
- 关键词自动回复
- 管理员命令：`#帮助` `#设置欢迎词` `#添加违禁词` `#删除违禁词` `#添加回复` `#删除回复`

### 完整版（邀请码激活）
- 签到 / 积分 / 排行榜
- 踢人 / 拉黑 / 管理员权限
- 自动审核 / 活跃度统计
- 好友验证自动通过
- 心跳保活

## 安装

1. 将整个文件夹复制到 WAuxiliary 插件目录
2. 重启 WAuxiliary 或重新加载插件
3. 在群里发送 `#帮助` 查看可用命令

## 激活完整版

在群里发送：
```
#激活 你的邀请码
```

激活后自动下载完整版功能，重启后自动加载。

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.java` | BeanShell 插件入口 |
| `info.prop` | 插件元信息 |
| `bridge.dex` | JNI 桥接（完整版需要） |

## 管理员设置

首次使用请在插件设置中配置：
- **super_admins**: 管理员微信 ID（JSON 数组格式，如 `["wxid_xxx"]`）
- **server_url**: 服务器地址（激活完整版需要）
