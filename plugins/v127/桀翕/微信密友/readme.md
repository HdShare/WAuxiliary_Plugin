# 微信密友

作者：桀翕 | TG：@jiexi1314 | 交流群：https://t.me/jieximy

## 简介

WAuxiliary 子插件，将指定联系人的聊天会话、消息通知、搜索记录完全隐藏。

## 安装

将插件放入 `/storage/emulated/0/Android/media/com.tencent.mm/WAuxiliary/Plugin/微信/`，重启微信即可。

## 使用

1. 在任意聊天框发送 `密友设置` 进入配置
2. 开启「密友保护」总开关，添加需要隐藏的联系人
3. 发送 `//hide` 隐藏密友，发送 `//show` 恢复显示（可设置密码锁）

## 注意

- 朋友圈隐藏需手动在微信资料页开启「不看他/她朋友圈」
- 仅对当前微信账号生效
