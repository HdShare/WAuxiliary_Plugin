# 自定义通知脚本使用说明
# 此版本更新说明:
1. 通知栏标记已读
   在通知栏操作按钮中新增「标记已读」，与现有「回复」按钮并列。点击后将对应会话标记为已读并自动关闭通知，交互体验与 Telegram 一致。
   该功能可在每个会话的单独配置中开启，亦支持通过批量修改统一设置，默认关闭。
2. 全局通知图标自定义
   在插件主设置页新增「通知图标」入口，支持从相册选择任意图片作为通知栏小图标。选定图片将自动保存至插件目录，路径持久化存储，重启后生效。未设置时沿用原有默认图标。
## 📖 脚本概述

这是一个会话级自定义通知脚本，支持按私聊/群聊单独配置通知行为。

## 🚀 快速开始

在聊天窗口输入 **通知设置** 点击发送，或点击 **插件设置**，即可打开设置界面。

## 🔧 主要功能

- 会话级开关：每个私聊/群聊独立启用或关闭规则
- 免打扰模式：不弹通知
- 通知震动、铃声：每个会话独立设置
- 快捷回复：每个会话独立开关
- 时段静默：开始/结束时间使用时间选择器设置
- 群聊规则：
  - 屏蔽 `@所有人`
  - 屏蔽 `@我`
  - 仅显示指定成员通知（白名单）
  - 屏蔽指定成员通知（黑名单）

## 🧩 配置入口

- 主入口：`通知设置`
- 会话规则：`会话列表与规则`
- 会话详情：点击具体会话进入配置

## 💡 使用建议

- 先给重要会话开启接管，再逐步微调规则
- 群聊优先配置成员白名单/黑名单，减少无效通知
- 若修改铃声后未生效，可重载插件或重启微信再测试

## ⚠️ 注意事项

- 快捷回复依赖系统通知输入框，需保证微信通知权限正常
- 不同 ROM 的通知策略可能影响铃声/震动表现
