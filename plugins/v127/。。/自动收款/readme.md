## 1.3更新说明
- 新增随机延迟区间，可为每笔收款随机等待指定毫秒范围后再执行
- 新增每日单日收款总结，可定时发送到多个好友或群聊
- 每日总结包含当天收款笔数、总金额、付款人统计和最近明细
- 新增手动发送总结：在聊天框发送“转账收款总结”即可发送到当前会话
- 自动定时总结发送成功后会清理当天统计；手动发送总结不会清理数据

## 🚀 开启步骤
1. 任意聊天框发送“**转账设置**”进入配置页面
2. 找到并开启“**自动收款**”开关
3. 根据您的需求，完成各项细化设置
4. 需要临时查看总结时，在目标会话发送“**转账收款总结**”

## ⚙️ 功能详解
| 功能模块 | 说明 |
|:----------|:------|
| **名单管理** | 支持设置白名单/黑名单，精准控制收款对象 |
| **金额识别** | 可按设定金额范围自动接收或拒收转账 |
| **关键词过滤** | 智能识别转账备注，匹配关键词自动处理 |
| **随机延迟收款** | 可自定义收款延迟区间，每笔收款随机等待后执行 |
| **自动退款** | 拒收后支持原路自动退回，无需手动操作 |
| **到账回复** | 收款成功后自动向对方发送确认消息 |
| **每日总结** | 支持每天固定时间向多个好友/群聊发送当天收款统计 |
