# 全局变量

::: warning 警告
本文档适用于 WAuxiliary v1.2.6 版本
:::

## 变量说明

- `hostContext`  
  宿主上下文

- `hostVerName`  
  宿主版本名

- `hostVerCode`  
  宿主版本号

- `hostVerClient`  
  宿主客户端

- `moduleVer`  
  模块版本

- `cacheDir`  
  缓存目录

- `pluginDir`  
  插件目录

- `pluginId`  
  插件标识

- `pluginName`  
  插件名称

- `pluginAuthor`  
  插件作者

- `pluginVersion`  
  插件版本

- `pluginUpdateTime`  
  插件更新时间
