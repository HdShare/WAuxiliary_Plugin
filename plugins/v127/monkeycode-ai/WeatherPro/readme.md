# 天气Pro

和风天气 API 驱动的微信天气查询插件，支持实时天气、多日预报、订阅推送和 UI 设置面板。

## 功能

| 功能 | 命令 | 说明 |
|------|------|------|
| 实时天气 | `天气 北京` | 查询当前温度/体感/风向/湿度/能见度/气压 |
| 三日预报 | `预报 北京` | 查看未来 3 天天气详情（含日出日落） |
| 订阅推送 | `订阅 北京` | 每日定时推送天气到系统通知栏 |
| 取消订阅 | `取消订阅` | 取消所有城市订阅 |
| 查看订阅 | `我的订阅` | 查看已订阅城市和推送设置 |
| 设置面板 | `天气设置` | 通过消息命令管理配置 |
| UI 面板 | 在 WAuxiliary 插件管理点击设置 | 可视化设置界面 |
| 帮助 | `天气帮助` | 显示完整命令列表 |

## 配置命令

```
天气设置 key <API Key>             设置和风天气 Key
天气设置 推送时间 <HH:mm>           设置推送时间（默认 08:00）
天气设置 推送 开/关                  开关每日推送
天气设置                            查看当前配置
```

## 获取 API Key

1. 访问 [和风天气控制台](https://dev.qweather.com)
2. 注册并创建免费项目
3. 获取 API Key（免费版每天 1000 次调用）
4. 在插件中设置 Key
