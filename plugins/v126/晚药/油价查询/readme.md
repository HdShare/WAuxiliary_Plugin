微信油价查询插件 - 使用说明

安装与加载

1. 将 油价插件文件 放入插件文件夹（文件夹名随意，如 油价查询）
2. 将整个插件文件夹放入 WAuxiliary 的插件目录
3. 在 WAuxiliary 模块中重新加载插件即可生效

使用方法

在微信任意聊天窗口（好友/群聊）发送以下格式指令：

油价 [地区名]

示例：

· 油价 北京
· 油价 云南
· 油价 广东

查询范围

目前支持查询全国各省/直辖市/自治区的油价，包括但不限于：

北京、天津、上海、重庆、河北、山西、辽宁、吉林、黑龙江、江苏、浙江、安徽、福建、江西、山东、河南、湖北、湖南、广东、广西、海南、四川、贵州、云南、西藏、陕西、甘肃、青海、宁夏、新疆、内蒙古

返回内容

```
⛽️ 云南省 油价 (2026-05-09)
━━━━━━━━━━━━━━
92号汽油  8.86 元/升
95号汽油  9.51 元/升
98号汽油  10.19 元/升
0号柴油   8.47 元/升
89号汽油  8.16 元/升
━━━━━━━━━━━━━━
```

89号汽油部分省份无数据时将自动隐藏，不显示该行。

注意事项

· 指令必须以“油价”开头，地区名支持模糊匹配
· 如输入 油价 后不加地区名，会收到格式提示
· 如地区名输入错误或无数据，会收到相应错误提示
