# AI 智能绘图与编辑插件 使用说明书 (WAuxiliary Plugin)

本插件是为 WAuxiliary 框架设计的 AI 绘图与编辑插件，通过调用中转 API（兼容 Gemini、Imagen 系列模型），在微信内提供文生图与多图融合编辑功能。

---

## 🌟 核心特性
*   **文生图 (Text-to-Image)**：输入描述词，自动生成对应的画面。
*   **多参考图融合与变图 (Multi-Reference Image Fusion)**：
    *   支持连续保存聊天窗口内的多张参考图片。按顺序发送图片后发送编辑指令，插件将多张图片等比缩放并注入 API 载荷中进行风格与角色融合。
    *   采用先进先出（FIFO）机制管理参考图队列，队列上限可配置。
    *   **5分钟时效限制**：参考图片加入队列后开始计时，5分钟内若未发送编辑指令，该图片将自动从队列中过期释放，防止后续遗留图片干扰创作。
    *   **自动清空**：图片处理并发送成功后，自动清空当前会话的参考图缓存队列，防止后续生图受历史残留图片干扰。
*   **参考图上限配置**：可在设置面板中调整参考图收集上限（支持：1张、2张、3张、4张、5张、10张）。
*   **比例与分辨率计算 (1K/2K/4K)**：内置分辨率与比例计算逻辑，支持 1K、2K、4K 档位配置。
*   **「自动比例」识别与格式处理 (Auto Aspect Ratio)**：
    *   图生图模式下选择“自动”比例时，插件会通过 BitmapFactory 读取并解析最新发送的参考图片的物理宽高比。
    *   自动将配置项转换为接口支持的标准格式（如 16:9、1:1 等），避免接口报错。
*   **内存优化与大图压缩 (OOM 防御)**：
    *   等比下采样：采用 BitmapFactory 的 inSampleSize 采样技术，在解码阶段将大图等比缩放至最大长边 2048 像素内。
    *   高压 JPEG 转换：以 80% 质量压缩为 JPEG，将文件体积缩减至 100~300KB，降低 Base64 编码的内存占用，防止堆内存溢出（OutOfMemoryError）。
*   **异步文件状态检测与轮询**：
    *   自动过滤 `THUMBNAIL_DIRPATH://` 虚拟路径，确保只处理有效的物理文件。
    *   后台线程轮询等待机制：若微信接收原图较慢，插件以 200ms 的频率轮询等待文件写入（最长 5 秒），写入完成后计入缓存。
*   **缓存与记录清理 (Clear Cache & Logs)**：
    *   指令清除：支持主人发送 `#清除日记`、`#清除缓存` 或 `#清除日志` 进行清理。
    *   UI清除：设置弹窗底部提供「清除缓存」按钮，支持手动清理。
    *   功能：删除本地临时缓存的图片，并重置内存中的参考图队列。
*   **即时生效**：支持配置文件热加载，在设置面板修改保存后配置立即生效，无需重启微信。

## 🛠️ 第一步：配置您的中转 API (修改 `config.prop`)

可在微信聊天窗口发送 `ai绘图设置` 唤起设置面板进行设置，或直接修改 `config.prop` 文件：

```properties
# 1. 中转 API 基础地址
api_base_url=https://yunwu.ai

# 2. 中转 API 密钥 (替换为实际 Key)
api_key=sk-xxxxxxxxxxxxxxxxxxxxxxxx

# 3. 统一 AI 绘图与编辑设置 (文生图与图生图共用相同配置)
model=gemini-3.1-flash-image-preview
resolution_level=1K
aspect_ratio=自动
size=1024x1024

# 变图参考图缓存上限数 (支持: 1, 2, 3, 4, 5, 10)
ref_image_count=1

# 4. 指令触发词
trigger_word=#生图
edit_trigger_word=#编辑图
```

---

## 📲 第二步：安装与加载插件

1. 将整个文件夹复制到手机存储中 **WAuxiliary 的插件存放目录**下。
2. 打开微信 -> 进入 **WAuxiliary 设置** -> **插件设置** -> **本地插件**。
3. 在本地插件列表中启用 **AI智能绘图与编辑 (1.3.0)** 即可。

---

## 💬 第三步：使用指令与功能演示

### 1. 文生图 (触发词：`#生图`)
直接输入触发词加描述词。
*   **发送指令**：
    > `#生图 一只穿着宇航服、在太空漂浮的柯基犬`
*   **高级动态模型切换 (仅限主人)**：
    > `#生图-gemini-3-pro-image-preview 一只绿色的兔子`
    *   *(注：安全机制仅限主人使用自定义模型。第三方用户发送该指令会被自动拦截，并降级为主人配置的默认模型。)*

---

### 2. 智能多图融合编辑与变图 (触发词：`#编辑图`)
插件会记住当前聊天窗口最新的多张参考图。
*   **操作步骤**：
    1. 在微信里发送两张或多张图片。
    2. 紧接着发送指令进行编辑/融合：
       > `#编辑图 请将这两张图片进行光影和风格融合，人物站立在风景背景中`
*   **异步避让机制**：
    若图片尚在下载中，发送指令后机器人会返回提示：`【AI提示】最新参考图正在高速下载中，请在 3-5 秒后重新发送指令。`。
*   **👥 群聊成员级独立隔离说明**：
    在群聊中，插件已实现**微信 ID 级绝对隔离**。
    *   **独立队列**：群员 A 发图只会存入 A 的专属队列，群员 B 发图存入 B 的专属队列，作画时仅读取命令发送人自己的队列，**不同成员在同一个群内并发作画完全互不干扰，彻底防冲突与盗图**。
    *   **时效性自释放**：每个成员的队列均受双重时效控制：**5分钟内若未发送编辑指令，或群内该成员的后续消息数超过滑窗限制，图片缓存将自动过期并安全清空释放**。
*   **高级动态模型切换 (仅限主人)**：
    > `#编辑图-gemini-2.5-flash-image 变成水墨风格`

---

### 3. 一键清除缓存与日记 (指令：`#清除日记` / `#清除缓存` / `#清除日志`)
*   **使用目的**：释放手机存储，或清空当前窗口的参考图缓存队列。
*   **安全校验**：仅限主人发送有效。
*   **效果**：清除所有在 `cacheDir` 下的临时图片，并清空内存中的参考图队列。
*   **发送指令**：
    > `#清除日记` *(或 `#清除缓存` / `#清除日志`)*

---

## ⚙️ 第四步：微信内置设置面板 (指令：`ai绘图设置`)

在微信聊天窗口发送指令呼出设置面板：
> **ai绘图设置**
*(安全校验：仅限主人账号发送时触发弹出，其他人发送无效)*

设置面板支持以下配置：
1. **更新中转 API Key 与 URL**。
2. **修改分辨率等级 (1K / 2K / 4K)**。
3. **配置“参考图缓存上限数”以启用多图融合**。
4. **配置“参考图时效范围限制”，可手动设置仅从最近的 5 条、10 条、15 条、20 条、30 条消息中提取图片，或不限制时效**。
5. **选择画面比例 (支持自动比例、1:1、9:16、16:9、21:9 等多个标准比例以及手动自定义模式)**。
6. **一键「清除缓存」：清空本地临时缓存和内存队列。**
7. **保存后立即生效。**
