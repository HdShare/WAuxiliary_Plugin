# WA 历史消息过滤器

一个用于 WAuxiliary 的插件，可以搜索和过滤微信历史消息，并支持导出和下载图片。

## 功能特点

- ✅ 按时间范围搜索消息（天/小时）
- ✅ 按消息类型过滤（文本/图片/语音/视频/文件/表情/链接）
- ✅ 按关键词搜索
- ✅ 按发送者过滤
- ✅ 区分自己和他人的消息
- ✅ 导出搜索结果为文本文件
- ✅ 图形化配置界面（输入 `/filter` 无参数）
- ✅ 从服务器下载图片到本地
- ✅ 所有命令不会出现在聊天记录中

## 使用方法

在任意聊天窗口的输入框中输入命令（命令不会发送出去）：

### 图形化界面（推荐）

```
/filter                 # 显示图形化配置界面，可视化配置所有选项
```

### 基本命令

```
/filter --help          # 显示帮助信息
/filter --reload        # 重载插件
```

### 搜索命令

```
/filter -d 7            # 搜索最近7天
/filter -h 24           # 搜索最近24小时
/filter -n 50           # 最多返回50条
```

### 过滤条件

```
/filter -k 会议         # 包含"会议"关键词
/filter -t image        # 只看图片
/filter -s wxid_xxx     # 指定发送者
/filter --self          # 只看自己的消息
/filter --others        # 只看别人的消息
```

### 组合使用

```
/filter -d 7 -k 通知 --others          # 最近7天别人发的包含"通知"的消息
/filter -h 48 -t image --save          # 最近48小时的图片并下载
/filter -t text -k 会议 -n 20          # 包含"会议"的文本消息，最多20条
/filter -k 照片 -ic 10 --save          # 包含"照片"的消息，并下载发送者前后10分钟的图片
```

### 图片上下文功能

当你使用关键字搜索时，可以使用 `-ic` 参数收集发送关键字消息的人在前后指定时间内发送的所有图片：

```
/filter -k 产品图 -ic 5 --save
```

这个命令会：
1. 查找包含"产品图"关键词的消息
2. 对于每条匹配的消息，找到发送者
3. 收集该发送者在这条消息前后 5 分钟内发送的所有图片
4. 自动去重
5. 下载这些图片（配合 `--save` 使用）

**使用场景：**
- 有人说"我发几张照片给你"，然后发了多张图片
- 有人说"产品图如下"，然后连续发了好几张图
- 通过关键字找到相关对话，批量下载前后发送的图片

## 参数说明

| 参数 | 简写 | 说明 | 示例 |
|-----|------|------|------|
| --days | -d | 最近N天 | -d 7 |
| --hours | -h | 最近N小时 | -h 24 |
| --count | -n | 最多返回N条 | -n 50 |
| --keyword | -k | 关键词搜索 | -k 会议 |
| --sender | -s | 指定发送者 | -s wxid_xxx |
| --type | -t | 消息类型 | -t image |
| --image-context | -ic | 图片上下文时间（分钟） | -ic 10 |
| --self | | 只看自己 | --self |
| --others | | 只看他人 | --others |
| --save | | 下载图片 | --save |
| --reload | | 重载插件 | --reload |
| --help | | 帮助信息 | --help |

### 消息类型

- `text` - 文本消息
- `image` - 图片
- `voice` - 语音
- `video` - 视频
- `file` - 文件
- `emoji` - 表情
- `link` - 链接

## 输出

- 搜索结果会保存为文本文件到插件目录
- 结果预览会发送到文件传输助手
- 使用 `--save` 参数时，图片会下载到 `media_<时间戳>` 目录

## 安装

1. 确保已安装 WAuxiliary
2. 将插件文件放到 WAuxiliary 插件目录
3. 重启微信或重载插件

## 图片下载说明

**重要提示：** 图片下载功能是从微信 **CDN 服务器重新下载**，而非从本地缓存提取。

**原因：**
- WA（WeChat Auxiliary）框架目前 **没有提供** 直接访问本地图片缓存的 API
- 其他框架（如 FK）提供了 `getImagePath(md5)` API 可以直接访问本地缓存
- 本插件使用 WA 提供的 `downloadImg()` API，该 API 会从 CDN 重新下载图片

**影响：**
- ✅ 优点：可以下载任何历史图片，不受本地缓存限制
- ⚠️ 注意：需要网络连接，下载速度取决于网络状况
- ⚠️ 注意：如果图片已过期或被删除，可能无法下载

**如果你需要更快的本地图片访问，可以考虑使用 FK 版本（计划中）。**

## 限制

- **图片下载**：✅ 完全支持（从 CDN 服务器下载）
- **文件/视频/语音下载**：❌ 暂不支持（微信使用专有协议，WA 未提供相关 API）
- **本地图片缓存访问**：❌ WA 框架不支持（其他框架如 FK 支持）

## 技术说明

本插件使用 WAuxiliary 提供的 API：
- `queryHistoryMsg()` - 查询历史消息
- `downloadImg()` - 从 CDN 下载图片（非本地缓存）
- `sendText()` - 发送消息到文件传输助手
